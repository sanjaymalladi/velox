# Getting Started

Velox makes it easy to create professional motion graphics programmatically. Follow these steps to get up and running quickly.

## Installation

### Global Installation

Install the Velox CLI globally using npm or pnpm:

```bash
npm install -g @velox-video/cli
# or
pnpm install -g @velox-video/cli
```

### Local Installation

For project-specific usage:

```bash
npm install @velox-video/cli
# or
pnpm add @velox-video/cli
```

## Create Your First Project

Create a new project instantly with the Velox CLI:

```bash
npx velox-video new my-video
cd my-video
```

This will create a new project with the following structure:

```
my-video/
├── src/
│   └── index.ts      # Your animation code
├── package.json
├── tsconfig.json
├── velox.config.ts   # Velox configuration
└── README.md
```

## Development Workflow

### 1. Write Your Animation

Create your animation in `src/index.ts`:

```typescript
import { canvas, rect, color, animate, text, group } from '@velox-video/core'

export default canvas({
  width: 1920,
  height: 1080,
  fps: 30,
  duration: 5.0,
  background: '#0f0f0f',
  children: [
    group({
      children: [
        rect({
          width: 200,
          height: 200,
          fill: color('red'),
          x: animate({
            from: 0,
            to: 100,
            duration: 2.0,
            easing: 'easeInOut',
          }),
          y: animate({
            from: 0,
            to: 100,
            duration: 2.0,
            delay: 0.5,
          }),
        }),
        text({
          content: 'Hello Velox!',
          font: 'bold 48px sans-serif',
          fill: color('white'),
          x: animate({ from: 0, to: 200, duration: 1.0 }),
        }),
      ],
    }),
  ],
})
```

### 2. Preview Your Animation

Run the local studio against your project:

```bash
npx velox-video preview src/index.ts
```

This will start a development server with Hot Module Replacement (HMR). Any changes you make to your animation code will be automatically reflected in the preview.

**Studio Features:**
- Real-time preview with HMR
- Playback controls (play, pause, stop)
- Speed adjustment (0.1x to 4x)
- Frame-by-frame navigation
- Console logging
- Performance metrics

### 3. Render Your Video

Export your project to a high-quality video file:

```bash
# Render with default settings
npx velox-video render src/index.ts --output my-animation.mp4

# Render with high quality
npx velox-video render src/index.ts --quality high --output my-animation.mp4

# Render to GIF
npx velox-video render src/index.ts --format gif --output animation.gif
```

## Interactive Code Playground

Try these interactive examples to learn Velox:

**Example 1: Basic Animation**
```typescript
import { canvas, rect, color, animate } from '@velox-video/core'

export default canvas({
  width: 1920,
  height: 1080,
  fps: 30,
  duration: 5.0,
  children: [
    rect({
      width: 200,
      height: 200,
      fill: color('blue'),
      x: animate({ from: 0, to: 200, duration: 3.0 }),
    }),
  ],
})
```

**Example 2: Multiple Elements**
```typescript
import { canvas, circle, color, animate, group } from '@velox-video/core'

export default canvas({
  width: 1920,
  height: 1080,
  children: [
    group({
      children: [
        circle({
          radius: 50,
          fill: color('red'),
          x: animate({ from: 100, to: 500, duration: 2.0 }),
        }),
        circle({
          radius: 30,
          fill: color('green'),
          x: animate({ from: 100, to: 500, duration: 2.0, delay: 0.5 }),
        }),
        circle({
          radius: 40,
          fill: color('yellow'),
          x: animate({ from: 100, to: 500, duration: 2.0, delay: 1.0 }),
        }),
      ],
    }),
  ],
})
```

**Try It Yourself:**

```typescript
// Modify this code and see the output:
import { canvas, rect, color, animate } from '@velox-video/core'

export default canvas({
  width: 1280,
  height: 720,
  children: [
    rect({
      width: 150,
      height: 150,
      fill: color('purple'),
      x: animate({ from: 0, to: 400, duration: 2.0 }),
    }),
  ],
})
```

**Expected output:** A purple square animating from left to right over 2 seconds.

## Common Commands

| Command | Description | Example |
|---------|-------------|---------|
| `velox-video new <name>` | Create a new project | `velox-video new intro-video` |
| `velox-video preview <file>` | Start development preview | `velox-video preview src/index.ts` |
| `velox-video render <file>` | Render to MP4 (default) | `velox-video render src/index.ts` |
| `velox-video render <file> --output out.mp4` | Custom output path | `velox-video render src/index.ts --output final.mp4` |
| `velox-video render <file> --format gif` | Render to GIF | `velox-video render src/index.ts --format gif` |
| `velox-video render <file> --quality high` | High quality render | `velox-video render src/index.ts --quality high` |
| `velox-video studio` | Start studio without file | `velox-video studio` |

## Configuration

Create a `velox.config.ts` file in your project root:

```typescript
import { defineConfig } from '@velox-video/core'

export default defineConfig({
  fps: 60,
  width: 1920,
  height: 1080,
  background: '#000000',
  output: {
    quality: 'high',
    format: 'mp4'
  },
  preview: {
    port: 3000,
    host: 'localhost',
    open: true
  }
})
```

## Environment Variables

| Variable | Description | Default |
|----------|-------------|---------|
| `VELOX_FPS` | Frames per second | 30 |
| `VELOX_WIDTH` | Output width | 1920 |
| `VELOX_HEIGHT` | Output height | 1080 |
| `VELOX_LOG_LEVEL` | Logging level | 'info' |
| `VELOX_PREVIEW_PORT` | Preview server port | 3000 |
