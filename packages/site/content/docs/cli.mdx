# CLI Reference

The Velox CLI is intentionally small. It covers the core workflow: scaffold a project, preview it locally, and render a final artifact.

## Installation

```bash
npm install -g velox-video
# or
pnpm add -g velox-video
```

## Commands

### `velox new <name>`

Scaffold a new project directory with a starter template.

```bash
velox new my-video
velox new my-video --template llm-explainer
```

Available templates currently include:

- `news-intro`
- `product-launch`
- `data-story`
- `kinetic-text`
- `particle-system`
- `llm-explainer`

### `velox preview <file>`

Start the local preview studio with hot reload.

```bash
velox preview video.ts
```

What it does:

- Loads the same `video.ts` entrypoint used for final render
- Opens the browser automatically
- Watches the source file and pushes config updates over WebSocket
- Lets you scrub, play, and export from the preview UI

### `velox render <file>`

Render a final artifact from the current entry file.

```bash
velox render video.ts --output result.mp4
velox render video.ts --format gif
velox render video.ts --format png-sequence --output frames
```

Supported formats:

- `mp4`
- `gif`
- `png-sequence`

## Recommended Workflow

1. Scaffold or write a `video.ts` file.
2. Run `velox preview video.ts` while iterating.
3. Tune layout, timing, and typography.
4. Run `velox render video.ts --output result.mp4` for final output.

## LLM-First Workflow

For local or hosted models, the shortest reliable path is to use the high-level schema API:

```ts
import { createExplainerVideo } from '@velox-video/core'

export default createExplainerVideo({
  title: 'How AI Agents Work',
  duration: 60,
  aspectRatio: '9:16',
  theme: 'tech',
  sections: [
    { type: 'hook', heading: 'From Prompt to Workflow' },
    { type: 'problem', heading: 'Teams Lose Time', points: ['Manual work', 'Context switching'] },
    { type: 'process', heading: 'The Loop', steps: ['Input', 'Plan', 'Execute', 'Review'] },
    { type: 'stats', heading: 'Impact', stats: [{ label: 'Time Saved', value: '68%' }] },
    { type: 'cta', heading: 'Start With Structure' },
  ],
})
```

## Troubleshooting

| Issue | Fix |
|-------|-----|
| `velox` not found | Reinstall the CLI globally or use `npx velox-video` |
| Preview loads but looks wrong | Check size, fps, and scene timing first |
| Render fails | Run the same file through `preview` first to isolate config issues |
| Output path is confusing | Pass an explicit `--output` value |
