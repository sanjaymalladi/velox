# CLI Reference

The Velox CLI provides a comprehensive set of commands for creating, previewing, and rendering motion graphics projects.

## Installation

Install the Velox CLI globally:

```bash
npm install -g @velox-video/cli
# or
pnpm install -g @velox-video/cli
```

## Available Commands

### Project Management

| Command | Description | Options |
|---------|-------------|---------|
| `velox-video new <name>` | Create a new project | `-d, --dir <directory>` - Specify project directory |
| `velox-video init` | Initialize Velox in current directory | `--typescript` - Initialize with TypeScript config |

### Development

| Command | Description | Options |
|---------|-------------|---------|
| `velox-video preview <file>` | Start local development studio with HMR | `--port <number>` - Specify port, `--host <host>` - Network binding |
| `velox-video studio` | Start the preview studio without a specific file | `--port <number>` - Specify port |

### Rendering

| Command | Description | Options |
|---------|-------------|---------|
| `velox-video render <file>` | Render to video (default: MP4) | `--output <file>` - Output path, `--format <mp4|webm|gif>` - Output format |
| `velox-video render <file> --quality <low|medium|high>` | Render with quality settings | `--preset <string>` - Encoder preset, `--crf <number>` - Quality (18-28) |
| `velox-video render <file> --loop` | Loop the animation | `--count <number>` - Number of loops |

### Build & Test

| Command | Description | Options |
|---------|-------------|---------|
| `velox-video build` | Build the project for production | `--out-dir <dir>` - Output directory |
| `velox-video test` | Run tests (if configured) | `--coverage` - Generate coverage report |

### Help & Version

| Command | Description |
|---------|-------------|
| `velox-video --help` | Show help information |
| `velox-video --version` | Show version number |

## System Prompt Templates

### For Component Creation
```
You are creating a Velox component. Use these patterns:
- Import: from '@velox-video/core' import { canvas, rect, color, animate }
- Structure: export default canvas({ width, height, children: [...] })
- Animation: Use animate() for time-based properties
```

### For Animation Definitions
```
You are defining animations in Velox. Key patterns:
- Use group() for organizing multiple elements
- Animate properties: x, y, width, height, opacity, etc.
- Timing: from, to, duration, delay, easing
```

### For Configuration
```
You are configuring a Velox project. Include:
- fps, width, height settings
- output.quality and output.format
- preview settings (port, host, open)
```

## Configuration

Create a `velox.config.ts` file in your project root:

```typescript
import { defineConfig } from '@velox-video/core'

export default defineConfig({
  fps: 60,
  width: 1920,
  height: 1080,
  background: '#000000',
  output: {
    quality: 'high',
    format: 'mp4'
  },
  preview: {
    port: 3000,
    host: 'localhost',
    open: true
  }
})
```

## Environment Variables

| Variable | Description | Default |
|----------|-------------|---------|
| `VELOX_FPS` | Frames per second | 30 |
| `VELOX_WIDTH` | Output width | 1920 |
| `VELOX_HEIGHT` | Output height | 1080 |
| `VELOX_LOG_LEVEL` | Logging level | 'info' |
| `VELOX_PREVIEW_PORT` | Preview server port | 3000 |

## Interactive Code Testing

Try these interactive examples:

**Example 1: Basic Animation**
```typescript
import { canvas, rect, color, animate } from '@velox-video/core'

export default canvas({
  width: 1920,
  height: 1080,
  fps: 30,
  duration: 5.0,
  children: [
    rect({
      width: 200,
      height: 200,
      fill: color('red'),
      x: animate({ from: 0, to: 100, duration: 2.0 }),
      y: animate({ from: 0, to: 100, duration: 2.0, delay: 1.0 }),
    }),
  ],
})
```

**Expected output:** A red square animating from (0,0) to (100,100) over 2 seconds.

**Example 2: Multiple Elements**
```typescript
import { canvas, circle, color, animate, group } from '@velox-video/core'

export default canvas({
  width: 1920,
  height: 1080,
  children: [
    group({
      children: [
        circle({
          radius: 50,
          fill: color('blue'),
          x: animate({ from: 100, to: 500, duration: 3.0 }),
        }),
        circle({
          radius: 30,
          fill: color('green'),
          x: animate({ from: 100, to: 500, duration: 3.0, delay: 1.0 }),
        }),
      ],
    }),
  ],
})
```

**Expected output:** Two circles (blue then green) animating horizontally with a 1-second delay between them.

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Command not found | Ensure `@velox-video/cli` is installed globally |
| Module not found | Run `pnpm install` in project directory |
| Rendering issues | Check WebAssembly support, try `--quality medium` |
| Preview not loading | Check port availability, firewall settings |

## Advanced Usage

### Programmatic Usage
```typescript
import { renderVideo } from '@velox-video/core'

renderVideo('./src/index.ts', {
  output: 'output.mp4',
  quality: 'high',
  format: 'mp4'
}).then(() => {
  console.log('Rendering complete!')
})
```

### Interactive Debugging
```bash
# Start studio for debugging
velox-video studio

# Then in another terminal, test render versions
velox-video render src/index.ts --quality low --output test.mp4
```
